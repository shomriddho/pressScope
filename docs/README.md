# pressScope Documentation

## Overview

pressScope is a Payload CMS 3.0 blank template designed as a starter for building web applications with Payload CMS integrated into a Next.js frontend. This template provides a pre-configured setup with authentication, media uploads, and a GraphQL API, making it ideal for rapid development of content-managed websites and applications.

The project combines the power of Payload CMS for content management with Next.js for the frontend, offering a type-safe, scalable solution for modern web development.
